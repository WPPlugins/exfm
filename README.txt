== Description ==

The exfm site player is the easiest way to play back music on your site - mp3's, SoundCloud, Bandcamp and more. It even works on iPhone and Android. It only takes a few seconds to set up.


== Installation ==

1. Unzip exfm.zip and upload the entire folder to wp-content/plugins.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Any link that ends in .mp3, any SoundCloud embeds and Bandcamp embeds will automatically show up in the exfm music player.
